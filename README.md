# Quiz-System-Interface-KBC-
Built a KBC-Like Interface For Hosting a Quiz At Doon University.
Implemented a option randomizer function so that all the options are not the same continuously.
Added Clocks Ticks and Rules Explanation.

# Key Controls for igniteX_v4-1.html:
Click M To Toggle Mute/Unmute.
Click End To Go Directly to Thank You page.
Click 1 to Switch to the Welcome page.
Click 0 For round 1 rules.
Click 9 For round 2 rules.
Click 8 For round 3 rules.
Click N to Switch to any question based on its genre which is colour coded.

# Features of igniteX_Round3.html:
Added a KBC-inspired Locking Mechanism and selecting mechanism , which outputs answer with a colour either greeen or red as per the correctness of the answer provided , with supporting sounds.
Click N to Switch to any question based on its genre which is colour coded.
